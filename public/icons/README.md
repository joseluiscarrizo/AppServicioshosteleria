# Iconos PWA — Camareros Bcn

Para una PWA completa, genera estos iconos desde el favicon:

## Generación automática con Sharp (Node.js):
```bash
npm install sharp -D
node scripts/generate-icons.js
```

## O manualmente:
- `icon-192.png` — 192x192px — para Android home screen
- `icon-512.png` — 512x512px — para Android splash screen y iOS

## Herramientas online:
- https://realfavicongenerator.net
- https://maskable.app (para iconos maskable)
